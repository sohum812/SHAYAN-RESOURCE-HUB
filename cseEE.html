EE-101

Introduction

  • Active Network: Contains energy sources (e.g., amplifiers, transistors). Example: A circuit with a battery.
  • Passive Network: No energy sources (e.g., resistors, capacitors, inductors). Example: RC filter circuit.
  • Linear Network: Follows superposition theorem (e.g., circuits with R, L, C). Example: Ohm’s law-based circuits.
  • Non-linear Network: Does not obey superposition (e.g., diodes, transistors). Example: LED circuit.
  • Unilateral Network: Current flows in one direction (e.g., diode circuits). Example: Rectifier.
  • Bilateral Network: Current flows both directions (e.g., transmission lines). Example: Resistor network.
  • Symmetrical Network: Identical behavior in both directions. Example: Balanced bridge circuit.
  • Unsymmetrical Network: Behavior varies with direction. Example: Unbalanced Wheatstone bridge.
  • Basic Circuit Elements: Resistors (limit current), Capacitors (store charge), Inductors (store energy in magnetic field).
  • Voltage Source: Provides constant voltage (e.g., battery). Example: 9V battery.
  • Current Source: Provides constant current (e.g., solar cell). Example: Ideal current source.
  • Independent Source: Fixed output (e.g., DC power supply).
  • Dependent Source: Output depends on another parameter (e.g., VCVS in op-amps). Example: Transistor model.
  • Kirchhoff’s Current Law (KCL): Sum of currents at a node is zero. Example: Parallel resistors.
  • Kirchhoff’s Voltage Law (KVL): Sum of voltages in a loop is zero. Example: Series resistors.
  • Common Symbols:
    • Resistor (R), Capacitor (C), Inductor (L)
    • Voltage source (V), Current source (I)
    • Ground symbol (⊥)
